About#

What is this product?#

A goal tracker that uses objectives, key results, and tasks to keep you accountable. A goal == an objective.

Each objective has associated key result(s). Each key result has associated task(s). Tasks can be configured at different recurrences: one-time, daily, weekly, yearly.

Users can review and check off their tasks for each day. By doing this, the user will have a daily reminder of what they're working towards.

More information about objective and key result (OKR)

Definitions#

  • Objective
    • A clearly defined goal
    • An objective will either be in a state of
      • 5 / 10 Total Tasks Completed: In-progress with number of associated tasks (cumulative for all associated key results) completed thus far
      • Objective (Action Required: Expand sections to see action required.): In-progress with action required
      • Done: Done
  • Key Result
    • A measurable success criteria used to track attainment of an objective. There can be many key results associated with one objective.
    • A key result will either be in a state of
      • 1 / 2 Total Tasks Completed: In-progress with number of associated tasks completed thus far (cumulative for all associated tasks)
      • Key Result (Action Required: Create an associated task): In-progress with action required; a task needs to be created for the key result
      • Key Result (Action Required: Set recurrence for associated task(s)): In-progress with action required; an associated task needs a recurrence set
      • Done: Done
  • Task
    • A concrete activity that needs to be accomplished to track attainment of a key result. There can be many tasks associated with one key result.
    • A key result will either be in a state of
      • Task (Action Required: Set Recurrence): In-progress with action required; a recurrence needs to be set
      • Task (One-Time): One-time task
      • Task (Recurring): Recurring task
      • Done: Done
  • Event
    • An event for a task. If a task is set to occur only one, only one event on the selected day will be created. If a task is set to recurr (ex: monthly from October 1, 2022 to December 1, 2022), multiple events for the task will be created (ex: October 1, 2022, November 1, 2022, and December 1, 2022).

Example#

  • Objective: Post more vlogs on YouTube (October 2022 - November 2022)
    • Key Result: Post one video per week between October 1, 2022 and November 31, 2022
      • Task: Brainstorm ideas once a month between October 1, 2022 and November 31, 2022
      • Task: Create publishing schedule
      • Task: Edit video 1x/week
      • Task: Publish video 1x/week
    • Key Result: Create a YouTube profile pic
      • Task: Brainstorm ideas once a month between October 1, 2022 and November 31, 2022
      • Task: Spike Test Canva
      • Task: Spike Test Procreate
      • Task: Publish video 1x/week
    • Key Result: Research and document trends
      • Task: Brainstorm ideas once a month between October 1, 2022 and November 31, 2022
      • Task: Look up viral videos 1x/week

Instructions#

Add an objective / goal #

  1. Navigate to Home.
  2. Navigate to the bottom of the page.
  3. In the input field, enter an objective / goal.
  4. Click Add objective / goal/

Edit an objective / goal #

  1. Navigate to Home.
  2. Navigate to the bottom of the page.
  3. Click Edit Objective/

Delete an objective / goal #

  1. Navigate to Home.
  2. Navigate to the bottom of the page.
  3. Click Delete Objective/

Add a key result#

  1. Navigate to Home.
  2. Navigate to the the objective you want to add a key result for. Expand the accordion if needed.
  3. In the input field, enter a key result.
  4. Click Add key result.

Edit a key result#

  1. Navigate to Home.
  2. Navigate to the the objective you want to edit a key result for. Expand the accordion if needed.
  3. Click Edit Key Result.

Delete a key result#

  1. Navigate to Home.
  2. Navigate to the the objective you want to delete a key result for. Expand the accordion if needed.
  3. Click Delete Key Result.

Add a task#

  1. Navigate to Home.
  2. Navigate to the the objective you want to add an objective for. Expand the accordion if needed.
  3. Navigate to the the key result you want to add a task for. Expand the accordion if needed.
  4. In the input field, enter a task.
  5. Set a recurrence for the task.

Set a recurrence for a task#

  1. Navigate to Home.
  2. Navigate to the the objective you want to add an objective for. Expand the accordion if needed.
  3. Navigate to the the key result you want to add a task for. Expand the accordion if needed.
  4. Navigate to the the task you want to add a task for. Expand the accordion if needed.
  5. If a recurrence has not been created, click "set a recurrence" or click "Update Task".
  6. A new page will load. On this page, select the desired recurrence. If a recurrence was already created for the task, the new set recurrence will override the previous recurrence.

Edit a task#

  1. Navigate to Home.
  2. Navigate to the the objective you want to edit an objective for. Expand the accordion if needed.
  3. Navigate to the the key result you want to edit a task for. Expand the accordion if needed.
  4. Click Edit Task.

Delete a task#

  1. Navigate to Home.
  2. Navigate to the the objective you want to delete an objective for. Expand the accordion if needed.
  3. Navigate to the the key result you want to delete a task for. Expand the accordion if needed.
  4. Click Delete Task.

View today's task#

  1. In the navigation bar, click "Today's Tasks"
